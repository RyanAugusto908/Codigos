Tipos de caixas

Temos as caixas do tipo:

box-level → Toda vez que se insere este tipo, ela vai para a próxima linha e ocupa o tamanho inteiro da tela.

inline-level → Quando é inserido este tipo de caixa ela não quebra a linha e só ocupa o espaço que foi destinado a caixa e o conteúdo continua logo após a caixa.

Box level ex:

div

h1 a h6

p

main

header

nav

article

footer

form

video

inline-level ex

span

Este também é um modelo inline-level. Este também continua sem quebrar a linha.
