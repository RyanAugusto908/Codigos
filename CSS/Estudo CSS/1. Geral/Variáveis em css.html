Variáveis em Css

Nesta atividade nós declaramos algumas variáveis em css.

Para isso nós utilizamos a pseudo-classe chamada (:root)

Estas variáveis são utilizadas para definir alguns itens que são utilizados em nossa página para facilitar a seleção.

Também auxilia na hora de manipular varias páginas.
