Nesta aula vamos aprender sobre css externo.

O que é CSS?

O CSS Cascading Style Sheets que é folhas de estilos em cascata.

Ele serve para deixar nossa página mais bonita, agradável de visualizar

Com o CSS podemos por exemplo trocar as cores de fundo, trocar a cor de uma letra, aumentar o tamanho de uma letra, posicionar imagens em nossa página e etc.

Aqui vamos fazer alguns exemplos.

Para visualizarmos na prática que estilos podemos utilizar.

A intenção é melhorar e deixar a página mais agradável.

Com isso a intenção é a retenção do usuário navegando em nossa página.

Não se esqueçam, nosso foco é:

Experiências do Usuário.

Qual a chance do usuário permanecer em nossa página, sendo que o visual da página não tem nenhum atrativo para reter o usuário, por isso vamos estilizar da melhor forma para fazermos com que a experiência do usuário seja inesquecível e que ele volte novamente em nossa página
