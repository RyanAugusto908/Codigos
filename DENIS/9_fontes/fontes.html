Fontes Baixadas

Nesta aula vamos trabalhar com fontes baixadas

Vamos baixar fontes dos sites

https://www.netfontes.com.br/

https://www.dafont.com/pt/

Para isso vamos aprender alguns tipos de format de fontes

opentype (otf)

truetype (ttf)

embedded-opentype

truetype-aat (apple advanced typografhy)

svg


Para inserirmos a fonte vamos seguir o passo a passo.

Logo abaixo do style vamos criar um @font-face → enter →Vamos para o site escolher a fonte e fazer o download →vamos criar uma pasta fontes junto com os arquivos html→Vamos extrair as fontes para esta pasta→ na url vamos entre aspas colocar 'fontes/ctrl+espaço' para selecionar o nome da fonte e enter→,logo em frente colocamos format('')→no font-family iremos colocar o nome da fonte (podemos cirar este nome)→depois é só utilizar no font-family
