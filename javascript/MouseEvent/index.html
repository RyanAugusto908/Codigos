Eventos de Mouse

Nesta aula vamos entender o que é um evento de mouse.

Eventos são ações disparadas pela interação dos usuários na página.

E a aplicação desses eventos tornam as páginas interativas e dinâmicas.

  • onclick → Disparado quando recebe um clique.
  • ondblclick → Disparado quando recebe um clique duplo.
  • onmouseover → Disparado quando o mouse está sobre o elemento.
  • onmouseout → Disparado quando o mouse sai do elemento.
  • onmousemove → Disparado quando o mouse se move dentro do elemento.
  • onmousedown → Quando o botão do mouse é pressionado.
  • onmouseup → Quando o botão do mouse é solto.
  • onfocus → Quando o elemento recebe o foco.
  • onblur → Quando o elemento perde o foco.
  • onchange → Quando o valor de um campo é alterado.
  • onkeypress → Quando uma tecla é pressionada e solta.
  • onkeydown → Quando uma tecla é pressionada.
  • onkeyup → Quando uma tecla é solta.
  • onreset → Quando um formulário é resetado.
